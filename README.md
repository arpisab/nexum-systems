# Nexum Systems — Starter Repo

A ready-to-use React + Vite starter for the **Nexum Systems** proposal site.

## Stack

- React 18
- Vite 5
- Plain CSS

## Start locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Suggested workflow

### Cursor + GitHub

1. Open the folder in Cursor.
2. Run `npm install`.
3. Start with `npm run dev`.
4. Commit and push to GitHub.

### GitHub Pages

This repo is already suitable for static deployment after build.
For GitHub Pages you can either:

- deploy the `dist` folder with a GitHub Action, or
- connect the repo to a static hosting workflow.

### Hostinger

1. Run `npm run build`
2. Upload the contents of `/dist` to the target public folder on Hostinger.
